# SWW-Ecodan-integratie-Tibber
SWW planning met Ecodan en Tibber zodat verwarmen op goedkoopste momenten gaat. 

De integratie met Ecodan ESP en Tibber maakt het mogelijk om op goedkope uren SWW (Sanitair Warm Water) te maken en legionella preventie te draaien. Dit is puur mijn prive repo om zaken vast te gaan leggen. Maak zoveel code dat handig is als ik centraal ga vastleggen.
